# Requirement ID: FR_auto_1
- Description: The system shall provide a clear and visible indicator of its emotional support capabilities on the main screen.
- Source Persona: Alex
- Traceability: Derived from review group A1
- Acceptance Criteria: Given the user opens the main screen, when the screen finishes loading, then the interface shows a clear and visible indicator of its emotional support capabilities.
